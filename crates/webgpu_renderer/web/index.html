Kataglyphis WebGPU + glTF demo — checking WebGPU support…

WebGPU is not available in this browser

This demo needs the WebGPU API. It is available in Chrome/Edge 113+, Firefox 141+, and Safari 26+. On Linux you may need to enable chrome://flags/#enable-unsafe-webgpu.

The same renderer also runs natively on Vulkan/DX12/Metal: cargo run -p kataglyphis_webgpu_renderer --example viewer
